Area of Interests

Machine Learning

The term machine learning implies machines that are learning. Machine Learning or ML is a subset of Artificial Intelligence.
It allows the machines to learn from previous experiences which could be empirical data and make autonomous decisions based on these experiences.
In basic technical terms, machine learning uses algorithms that take empirical or historical data in, analyze it, and generate outputs based on that analysis.

Data Analytics

Data analytics is the science of analyzing data to make conclusions about that information.
Data analytics help a business optimize its performance, perform more efficiently, maximize profit, or make more strategically-guided decisions.
Data Analytics has an arsenal of software tools to acquire, store, process, visualize and report the data
